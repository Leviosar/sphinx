# Sphinx

Sphinx é um jogo multiplayer de perguntas e respostas, disponível para Android.

# Sphinx app

A pasta app é a *root* de um projeto [Flutter](https://flutter.dev/), sendo assim, para compilar a aplicação você vai precisar do Flutter instalado na sua máquina. Depois disso é só entrar na pasta e rodar `flutter pub get` para instalar as dependências e `flutter build` para compilar na plataforma padrão.
